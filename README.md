# Engenharia de Prompts e "Cicatrizes"
Nesta etapa, testei diferentes formas de extrair informações da IA para resolver problemas reais:

Prompt inicial: "Como funciona o State no React?"

Dificuldade (Cicatriz): A resposta foi muito teórica e não explicava como aplicar o estado em múltiplos componentes.

Refinamento (Troubleshooting): Ajustei o prompt para: "Explique como elevar o estado (lifting state up) com um exemplo prático de um contador".

Resultado: Consegui compreender a importância da imutabilidade e como os dados fluem de pai para filho.

# Miniguia de Estudo (Entrega Final)
Resumo: O React é uma biblioteca focada em componentes. O segredo está em dividir a interface em partes menores e reutilizáveis.

Glossário: * JSX: Sintaxe que permite escrever HTML dentro do JavaScript.

Props: Propriedades passadas para componentes (como argumentos de uma função).

State: Dados que mudam ao longo do tempo e fazem o componente reagir e atualizar.
